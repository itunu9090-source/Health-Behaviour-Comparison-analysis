# Health-Behaviour-Comparison-analysis
This project analyzes the health behaviour dataset to compare smokers and non-smokers

### Project Overview
This data analysis project aims to provide insights into the health behaviour comparison. By analyzing various aspects of the health data, we seek to identify rate at which smoker and smoker effect to health issues, make data-driven recommendations, and acquire a full knowledge of the health behaviour comparison.
